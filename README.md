Сборка Gulp от AiGen31
======================
В данном иструменте имеется всё необходимое для Frontend разработки

Ключевые возможности:
---------------------
* Предпроцессор Sass
* Сервер BrowserSync с SSI для встраивания HTML документов в друг друга
* Задачи для оптимизации и минификации файлов

Первый старт:
-------------
Для работы вам потребуется [Git](https://git-scm.com/) и [NoteJS](https://nodejs.org/en/)
1. Установите ПО для правильной работы инструмента
2. Инициализируете свой проект и следуете инструкции
`npm init`
3. Установите Gulp и Bower глобально
`npm i gulp -g`
`npm i bower -g`
4. Запустите gulp командой `gulp`
